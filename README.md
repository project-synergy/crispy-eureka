# crispy-eureka
Web Hosting Service

# Project Setup

1) Navigate to C:\Users\Admin\Documents\GitHub\crispy-eureka>

```
py -m pip install virtualenv
```

2) setup virtualenv 

```
py -m virtualenv .
```

3) Activate virtualenv

```
Scripts\activate.bat
```

4) Install requirements 

```
py -m pip install -r requirements.txt
```

5) Run app

```
cd src

py app.py
```

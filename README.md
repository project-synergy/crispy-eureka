# crispy-eureka
Web Hosting Service

# Project Setup

1) Navigate to crispy-eureka

```
cd crispy-eureka
```

2) Install virtualenv

```
py -m pip install virtualenv
```

3) setup virtualenv 

```
py -m virtualenv .
```

4) Activate virtualenv

```
Scripts\activate.bat
```

5) Install requirements 

```
py -m pip install -r requirements.txt
```

6) Run app

```
cd src

py app.py
```
